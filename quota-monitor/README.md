# Antigravity Quota Monitor

A local dashboard to check your AI usage limits.

## 🚀 Quick Start

1.  **Open two terminals** in this directory.

2.  **Terminal 1 (Backend):**
    ```bash
    npm install
    node server.js
    ```
    You should see: `Server running on port 3001`

3.  **Terminal 2 (Frontend):**
    ```bash
    cd client
    npm install
    npm run dev
    ```
    You should see: `Local: http://localhost:5173/`

4.  **Open Browser:**
    Go to [http://localhost:5173](http://localhost:5173)

## Troubleshooting
- Ensure Antigravity IDE is open.
- If it says "Waiting for connection", try reloading the page or check the backend terminal for errors.
